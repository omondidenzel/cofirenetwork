# Photos for Photo Runner Section

Add your JPEG or JPG files here to display them in the photo runner section on the homepage.

## How it works:
1. Add `.jpg` or `.jpeg` files to this folder
2. Name them sequentially: `photo-1.jpg`, `photo-2.jpg`, `photo-3.jpg`, etc.
3. The photo runner will automatically display them in a continuous scrolling marquee

## Image Requirements:
- **Format**: JPEG (.jpg) or JPG
- **Recommended size**: 640x400 pixels
- **Optimize**: Compress images for faster loading

## Example:
```
images/
  photo-1.jpg
  photo-2.jpg
  photo-3.jpg
  photo-4.jpg
  photo-5.jpg
  photo-6.jpg
```

The photos will scroll automatically on the homepage in the "Photos running" section.
